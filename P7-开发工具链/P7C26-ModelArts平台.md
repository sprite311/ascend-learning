# P7C26 — ModelArts 云平台：从开发到部署的一站式体验

## 1. 什么是 ModelArts

ModelArts 是华为云上的 **AI 开发平台**，提供从数据处理、模型训练、到部署上线的全流程管理能力。

如果你没有自己的昇腾硬件，通过 ModelArts 可以直接租用昇腾算力进行开发和部署。

---

## 2. 核心功能

| 功能 | 说明 |
|------|------|
| **Notebook** | 在线 Jupyter 环境，预装昇腾开发工具链 |
| **训练任务** | 提交分布式训练作业，平台自动调度资源 |
| **模型管理** | 版本管理、模型注册 |
| **推理服务** | 一键部署模型为 REST API 服务 |
| **自动学习** | 低代码 AI 开发（适合非专业用户） |
| **数据管理** | 数据集标注、清洗、存储 |

---

## 3. 典型使用流程

```
1. 创建 Notebook（选择带 NPU 的运行环境）
2. 在 Notebook 中调试代码
3. 代码调通后，提交为训练作业（自动分配多卡/多机）
4. 训练完成后，将模型注册到模型仓库
5. 一键部署为在线推理服务
6. 获得 API 地址，应用端直接调用
```

---

## 4. 使用建议

- **适合场景**：没有自有硬件、需要弹性算力、团队协作开发
- **注意事项**：云平台 UI 变化较快，以原理和流程为主，具体操作参考最新官方文档
- **成本控制**：按需使用，开发时用小规格实例，训练时用大规格实例
